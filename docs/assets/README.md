# kbeatz Logo Assets

This directory contains the kbeatz logo and related visual assets.

## Files

| File | Description |
|---|---|
| `kbeatz-logo.jpg` | Primary logo - the beatz lamps photo (1417x1063 px, JPEG) |
| `kbeatz-logo.svg` | SVG wrapper referencing kbeatz-logo.jpg for vector-context usage |
| `kbeatz-logo-dark.svg` | Dark-mode variant placeholder |

## Alt Text Reference

Use the following alt text values consistently across all contexts:

| Context | File | Alt text |
|---|---|---|
| README.md header | `kbeatz-logo.jpg` | `kbeatz logo` |
| GitHub Pages site header | `kbeatz-logo.jpg` | `kbeatz logo` |
| Favicon or small icon usage | `kbeatz-logo.jpg` | `kbeatz` |
| Social / Open Graph image | `kbeatz-logo.jpg` | `kbeatz - self-hosted music collection manager` |

For HTML `<img>` elements:
```html
<img src="docs/assets/kbeatz-logo.jpg" alt="kbeatz logo" width="300">
```

For Markdown:
```markdown
![kbeatz logo](docs/assets/kbeatz-logo.jpg)
```

## Size Guidelines

The logo is a 1417x1063 photograph. Recommended display sizes:

| Use | Width |
|---|---|
| README header | 300 px |
| Docs site header | 150 px |
| Full resolution | 1417 px |
